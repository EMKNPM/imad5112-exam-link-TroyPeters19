The application was developed using Kotlin in Android Studio. It demonstrates the use of arrays, loops, functions, screen navigation, data classes, input validation, and error handling.

The application allows users to:
Add, view and organise grocery items for their camp trip

The application contains three main screens:

Splash Screen
The Splash Screen is the first screen displayed when the application launches.

It includes:

Application name
Student name
Student number
Start button
Exit button

Main screen that includes
Total items packed text view
Add Gear button



Detailed Screen
The Detailed Screen is the main working screen of the application.

It includes:

A Spinner for each item
An input field for Category
An input field for quantity
An input field for Comments
A Save data button
A Total items button
A Back to Base button.
